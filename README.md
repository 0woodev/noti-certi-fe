1. 팀을 만드는 화면
2. 팀에 추가할 어플리케이션을 선택할 수 있어야 하고
3. 어플리케이션을 만드는 화면
4. 어플리케이션이랑 도메인을 연결하는 화면

5. 검색
    - 팀을 검색할 수 있음 -> 어플리케이션 선택
    - 어플리케이션 검색 -> 도메인 선택
    - 도메인으로 검색 -> 인증서 조회
    - SAN -> 인증서 -> 도메인들보고 -> 어플리케이션

6. 어플리케이션이나 팀, 도메인에 대해서 구독할 수 있어야 한다.

7. 팀을 조회하는 화면
